Busca

        
Busque na web e obtenha conteúdo completo.

        
Envie uma query e o Webiii retorna os resultados mais relevantes da internet, cada um com seu conteúdo pronto para uso pelas suas IAs.

      

      

        
Scrape

        
Dados prontos para LLMs.

        
Markdown, HTML, links e screenshots. Tudo com uma única chamada de API limpa e rápida.

      

      

        
Extração LLM

        
Estruturação inteligente.

        
Passe um JSON Schema e receba dados rigorosamente estruturados, potencializados pelo Google Gemini.

      

      

        
Mapeamento

        
Descubra todas as URLs.

        
Análise de sitemaps combinada com rastreio rápido. Encontre todas as páginas de um domínio em segundos.

      

      

        
Crawl

        
Rastreamento recursivo assíncrono.

        
Navegação BFS completa com geração de Job ID, polling, controle de profundidade e limite de páginas.

Renderização Inteligente

        
O motor do Playwright lida com JavaScript, SPAs e qualquer tipo de carregamento de conteúdo dinâmico automaticamente.

      

      

        

          
        

        
Markdown Perfeito

        
Converte HTML diretamente para Markdown formatado com GFM (tabelas e blocos de código), formato ideal para contexto de LLMs.

      

      

        

          
        

        
Limpeza de Conteúdo

        
O filtro Cheerio atua removendo menus de navegação, rodapés, popups, scripts e cookies, deixando apenas o que importa.

      

      

        

          
        

        
Capturas de Tela

        
Gere screenshots da página inteira ou apenas do conteúdo visível no topo, com a adição de um único parâmetro na requisição.

      

      

        

          
        

        
Leitura de Sitemaps

        
O mapeador entende e analisa hierarquias complexas de sitemap.xml, encontrando até as URLs mais escondidas da aplicação.

      

      

        

          
        

        
Segurança Integrada

        
Projetado para estar na web. Validação por Bearer Token e mitigação de DDoS através de Rate Limiting nativo via IP.

Pronto para construir?

      
Acesse e extraia dados da web com zero limitações. Self-hosted e focado no desenvolvedor.

      

        
Terminal

        
# Instale e inicie o servidor
cd "API Scraping"
npm install
npx playwright install chromium
npm run dev

# Faça sua primeira requisição
curl -X POST http://localhost:3002/v1/scrape \
  -H "Authorization: Bearer webiii-dev-key-2026" \
  -H "Content-Type: application/json" \
  -d '{"url":"https://example.com","formats":["markdown"]}'
